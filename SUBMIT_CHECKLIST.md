## Steps
1. **Join** the competition (accept rules) on the competition page.
2. **Publish the notebook:** upload `research_agent.ipynb` to Kaggle → run all cells so saved output shows it works → **Save/Publish** as **Public**. Copy its URL.
3. **Record the video** (~2 min) following `VIDEO_SCRIPT.md`; upload; copy the link.
4. **Fill the writeup:** paste `WRITEUP.md`, replace `<PASTE KAGGLE NOTEBOOK LINK>` and `<PASTE LINK>` with the notebook + video URLs.
5. **New Writeup** on the competition page → paste the writeup → set track **Freestyle** → **Submit**.
6. **Verify logged-out / incognito:** notebook opens public, video plays, writeup links resolve.
